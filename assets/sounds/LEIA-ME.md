# Som de virada de página

Coloque aqui um arquivo chamado `page-turn.mp3` (royalty-free).

Sugestão gratuita: freesound.org — busque "page turn" e filtre por licença CC0.
Arquivo ideal: ~0.5s, mono, ~50KB.

O código usa este arquivo automaticamente via Howler.js quando presente.
Se o arquivo não existir, usa síntese por Web Audio API como fallback.
